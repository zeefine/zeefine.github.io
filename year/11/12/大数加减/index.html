大数加减

Veröffentlicht am 2017-11-12 | in acm

给出2个大整数A,B，计算A+B的结果。

Input
第1行：大数A
第2行：大数B
(A,B的长度 <= 10000 需注意：A B有可能为负数）

Output
输出A + B

Input示例

68932147586
468711654886

Output示例

537643802472

解题思路
先用字符数组储存然后倒序转化为整数数组，结果输出时倒序输出，遇到0舍去

加法:进位问题，逢十进一

//假设第一个数大
for(i=0;i<len1;i++)
    {
        c[i]+=a[i]+b[i];
        if(c[i]>9)

            }

减法:退位问题，不够减时向前借位

//假设第一个数大
for(i=0;i<k;i++)
    {
        if(a[i]-b[i]<0)
        {
            c[i]+=10+a[i]-b[i];
            a[i+1]--;
        }

    }

在做减法时考虑到最后结果，分为大数减小数，小数减大数，两数相等3种情况
